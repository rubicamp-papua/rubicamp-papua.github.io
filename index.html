Belajar Coding di Akhir Pekan

Kelas Programming Akhir Pekan merupakan program belajar coding yang diselenggarakan di setiap akhir pekan (Jumat, Sabtu atau Minggu). Program ini kami kembangkan untuk memberikan kesempatan kepada siapa saja yang ingin belajar coding di sela-sela kesibukan hariannya baik itu bekerja, kuliah ataupun sekolah.

Alur Belajar Jelas

Kamu tidak perlu lagi kebingungan harus mulai dari mana dalam belajar coding. Kami sudah menyusun alur yang sangat jelas untuk membantumu belajar.

Belajar Di Akhir Pekan

Kamu tidak perlu meninggalkan aktivitas harianmu untuk mengikuti program ini. Kamu cukup meluangkan beberapa minggu di akhir pekanmu untuk mengikuti program ini.

Programming

Project Base
